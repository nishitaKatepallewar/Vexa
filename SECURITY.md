# Security Policy

## Supported Versions

Only the latest version of Vexa is supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by opening a [GitHub issue](https://github.com/nishitaKatepallewar/Vexa/issues).

Do not report security vulnerabilities through public GitHub issues or pull requests.

Include the following in your report:

- Description of the vulnerability
- Steps to reproduce the issue
- Potential impact of the vulnerability
- Any possible fixes (if known)

The reported vulnerabilities will be reviewed and addressed promptly.
